Elements

Elements are the building blocks of html. An element consists of an opening tag, content, and a closing tag. An opening tag includes the name of the element wrapped in opening and closing brackets. This indicates where the element begins. The content is the content of the element. The closing tag is the same as the opening tag but includes a forward slash. This indicates where the element ends.

Common Elements

Tag Element Description
<html> html The html element is the root element of an html document. It indicates the start of the html document.
<title>> title The title element contains the document title that is displayed in the browser tab.
<body>> body The body element contains the page content of the html document.
<paragraph> paragraph The paragraph element creates a paragraph or block of content.
<li> list The list element creates a list.
<ul> unordered list The unordered list element creates a bullet point list.
<ol> ordered list The ordered list element creates a list preceded by letters or numbers.
